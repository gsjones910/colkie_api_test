## Description

```bash
Language: TypeScript
Backend:  Nest framework
Database: Mongodb
Api doc:  Swagger
```

## Installation

```bash
$ npm install
```
Create a .env file from the .env.example file and enter the relevant parameters.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
